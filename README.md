## 2. winform 窗口程序

依赖：
pip install pyinstaller
打包：
打包单个exe文件: pyinstaller -F main.py
打包到一个目录中: pyinstaller -D main.py
打包不生产控制台窗口：pyinstaller --noconsole -F main.py


连接mysql数据库：
pip install mysql-connector-python

pip3 install mysql-connector
